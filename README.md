<h1>Capabilities</h1>
The repository is able to load and parse all the binaries from https://github.com/LunarG/SPIRV-Database.
<h1>Contributing</h1>
The files in the package spirv was generated by https://github.com/boredhuman/JavaSpirvGenerator. Contributions which are not located in the generated spirv directory will be accepted, otherwise they must be contributed to the generator repository.
